Why this stack wins for AI therapy:

Deepgram Nova-3 is the gold standard for STT in voice agents ‚Äî sub-150ms TTFT, $0.0043/min (cheapest by far), HIPAA compliant, and ranked #1 on the independent VAQI benchmark. The Flux model adds native end-of-turn detection so the AI knows when someone's finished speaking ‚Äî critical for natural conversation.

Claude Haiku (or Sonnet for more nuanced therapy responses) is the ideal LLM ‚Äî long context window for session memory, safe output defaults, and emotionally intelligent responses. Far better than GPT-4o-mini for therapeutic content. Pair with Anthropic's safety defaults.

ElevenLabs Flash v2.5 is the only TTS that sounds warm and genuinely human enough for a therapeutic context. At 75ms TTFB it's the fastest quality voice. The emotional range of ElevenLabs voices is unmatched ‚Äî this is the one non-negotiable premium spend. The 3,000+ voice library means you can offer users different voice personas.

For telephony infrastructure: use Vapi.ai or Retell AI as the orchestration layer (like Ash). They handle phone numbers, SIP, WebSockets, call routing natively. Or build directly on LiveKit + Twilio for full control.
Recommended architecture
Step 1 ‚Äî Telephony
Vapi.ai or Retell
WebRTC / SIP / PSTN
Handles inbound/outbound calls, phone number provisioning, call routing, barge-in detection. Acts like Ash ‚Äî users call a real number. $0.05‚Äì0.07/min overhead is worth the saved engineering time.
Step 2 ‚Äî Speech Recognition
Deepgram Nova-3
nova-3 + flux (streaming)
WebSocket streaming, ~150ms TTFT. $0.0043/min. End-of-turn detection with Flux model eliminates VAD latency. HIPAA BAA available. Handles accents, emotional speech, and noisy environments well.
Step 3 ‚Äî LLM Brain
Claude Haiku / Sonnet
claude-haiku-4-5 / claude-sonnet-4-5
200K token context for full session memory. Streaming responses for low latency. Haiku for speed+cost ($0.80/$4 per 1M), Sonnet for deeper empathy and nuance. System prompt defines the therapist persona, safety guardrails, and CBT/DBT frameworks.
Step 4 ‚Äî Voice Synthesis
ElevenLabs Flash v2.5
eleven_flash_v2_5
75ms TTFB, streaming synthesis. Choose a warm, calm female or male voice from the 3,000+ library. Custom voice cloning option lets you create a unique "therapist" persona voice. $0.050/1K chars.
Step 5 ‚Äî Session Memory
Memory Layer
Mem0 / Custom DB
Store session transcripts, user mood patterns, therapy progress, past topics. Feed summarized context into each new session via the Claude system prompt. Enables continuity across calls ‚Äî vital for therapy.
Step 6 ‚Äî Safety & Compliance
Guardrails
Crisis detection + HIPAA
Real-time keyword detection for crisis language ‚Üí escalate to human. Anthropic's built-in safety. HIPAA BAAs from Deepgram + ElevenLabs (enterprise). All audio encrypted in transit. Do not store raw audio; store only transcripts.
