Why this stack wins for AI therapy:

Deepgram Nova-3 is the gold standard for STT in voice agents ‚Äî sub-150ms TTFT, $0.0043/min (cheapest by far), HIPAA compliant, and ranked #1 on the independent VAQI benchmark. The Flux model adds native end-of-turn detection ‚Äî critical for natural conversation.

Claude Haiku (or Sonnet for more nuanced therapy responses) is the ideal LLM ‚Äî long context window for session memory, safe output defaults, and emotionally intelligent responses. Far better than GPT-4o-mini for therapeutic content.

ElevenLabs Flash v2.5 is the only TTS that sounds warm and genuinely human enough for a therapeutic context. At 75ms TTFB it's the fastest quality voice. The emotional range of ElevenLabs voices is unmatched ‚Äî this is the one non-negotiable premium spend.

For WebRTC browser calls: Use LiveKit Agents as the WebRTC layer ‚Äî open-source, battle-tested, and has first-class plugins for Deepgram + ElevenLabs + Anthropic. Your frontend uses the LiveKit JS SDK (5 lines), backend runs a Python agent that wires the full voice pipeline.

Prefer 1 provider? Go with ElevenLabs Conversational AI + Claude ($0.50/session). Their React SDK handles WebRTC natively ‚Äî connect Claude from their dashboard. Half the engineering, slightly higher cost.
Recommended architecture
Step 1 ‚Äî WebRTC Layer
LiveKit (open-source)
livekit-server + JS SDK
Handles WebRTC peer connections, TURN/STUN servers, NAT traversal, audio track routing, interruption events. Browser calls LiveKit room, backend agent joins same room. Self-host or use LiveKit Cloud (~$0.006/participant-min).
Step 2 ‚Äî Speech Recognition
Deepgram Nova-3
nova-3 + flux (streaming)
LiveKit's Deepgram plugin pipes audio directly. ~150ms TTFT. $0.0043/min. End-of-turn detection with Flux model. HIPAA BAA available. LiveKit plugin: livekit-plugins-deepgram
Step 3 ‚Äî LLM Brain
Claude Haiku / Sonnet
claude-haiku-4-5 / claude-sonnet-4-5
200K token context for full session memory. Streaming tokens. Haiku for speed+cost, Sonnet for deeper empathy. System prompt holds therapist persona, CBT/DBT frameworks, crisis guardrails, and summarized past sessions.
Step 4 ‚Äî Voice Synthesis
ElevenLabs Flash v2.5
eleven_flash_v2_5
75ms TTFB, streaming audio back through LiveKit room to browser. Warm, emotionally nuanced voice ‚Äî 3,000+ options. Custom voice cloning creates a unique therapist persona. LiveKit plugin: livekit-plugins-elevenlabs
Step 5 ‚Äî Session Memory
Memory Layer
Mem0 / Postgres + pgvector
Store session transcripts, mood patterns, therapy progress, past topics. Summarize at session end via Claude. Inject compressed summary into next session's system prompt. Enables true continuity ‚Äî vital for therapy.
Step 6 ‚Äî Safety & Compliance
Guardrails
Crisis detection + HIPAA
Parallel keyword classifier on transcript stream ‚Äî flags self-harm language and triggers escalation message + human alert. HIPAA BAAs from Deepgram + ElevenLabs (enterprise). Transcripts encrypted at rest. Never store raw audio.
